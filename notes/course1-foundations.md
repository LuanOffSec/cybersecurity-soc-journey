# Course 1 — Foundations of Cybersecurity

## Key Concepts

**CIA Triad:**
- Confidentiality: data accessible only to authorized users
- Integrity: data is accurate and unmodified
- Availability: systems and data accessible when needed

**CISSP 8 Security Domains:**
1. Security and Risk Management
2. Asset Security
3. Security Architecture and Engineering
4. Communication and Network Security
5. Identity and Access Management (IAM)
6. Security Assessment and Testing
7. Security Operations
8. Software Development Security

**Common Threat Actor Types:**
- APT (Advanced Persistent Threat): nation-state sponsored, 
  long-term stealthy access
- Hacktivists: ideologically motivated
- Insider threats: current or former employees with access
- Script kiddies: low skill, rely on existing tools

**Key Frameworks:**
- NIST CSF: Identify, Protect, Detect, Respond, Recover
- CIA Triad: foundation of all security decisions

## SOC Relevance
A SOC analyst operates primarily within the Detect and Respond 
phases of the NIST CSF. Understanding threat actor motivations 
helps prioritize which alerts require immediate escalation 
to the CSIRT team.
